# Sentinel Design System

## Philosophy

Sentinel follows a premium, minimal, product-first design language inspired by:

- Linear
- Vercel
- Stripe
- Apple
- Anthropic
- Notion

Every UI decision should prioritize clarity, whitespace, and professionalism.
